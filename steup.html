SETUP PROCESS

STEP BY STEP PROCESS IN SETTING UP GITHUB and VSCODE

  1. Download GitHub
  2. Download VSCode
  3. Download Git
  4. Connect Git through VSCode by terminal, initiating Git Bash. If everything is in order, the Git Bash should be colorful
  5. Setup Github Account
  6. Connect Github account to VScode

Make a folder of your desires

Make a repository (MAKE SURE ITS THE SAME NAME AS THE FOLDER FILE)

Use the terminal and make it Git Bash

Go through codemy.com/git/ and copy paste the instructions

Go through the repository in GitHub and copy paste the repository code

Reload and, if nothing went wrong, the repository should have the files you published

If you want to add more files, go through the source control and sync and commit changes

If you want other files, add in the file and add a file name while specifying if it is HTML, CSS, Python, Java, so on so forth by using .html, .css, .py
